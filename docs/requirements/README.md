# 📂 docs/requirements/

이 폴더는 e-커머스 백엔드 API의 **요구사항 및 정책 명세서**를 관리합니다.

각 도메인 또는 기능 단위로 파일을 분리하여 문서화하며,  
비즈니스 로직, 예외 처리 규칙, 정책 제한 등 정형화된 정보를 담는 것을 원칙으로 합니다.

> ❗ 아직 논의 중이거나 미확정된 내용은 GitHub Issue에서 먼저 관리 후 이곳에 반영합니다.
