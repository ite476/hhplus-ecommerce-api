# 📦 Product 도메인 요구사항

상품 정보 조회 및 인기 상품(판매량 상위 5개) 조회 기능에 대한 요구사항을 정의합니다.

---

## ✅ 기능 목록

### 1. 상품 목록 조회 API

- 전체 상품 목록을 조회할 수 있어야 한다.
- 각 상품 정보에는 다음 필드가 포함되어야 한다:
    - 상품 ID
    - 상품 이름
    - 가격
    - 현재 잔여 수량
- 조회 시점의 잔여 수량이 정확하게 반영되어야 한다.

### 2. 인기 상품 조회 API

- 최근 3일간 판매량 기준으로 **가장 많이 팔린 상품 5개**를 조회한다.
- 각 상품의 판매량도 함께 응답에 포함한다.
- 인기 상품 정보는 캐싱 또는 통계 테이블 등의 방식으로 성능 고려 가능

---

## ⚠️ 정책 및 제약사항

- **재고가 0인 상품**도 목록 및 인기 상품에 포함 가능 (표시만 잘 구분하면 됨)
- 인기 상품 집계는 주문 시점 기준으로 반영되어야 함
- 성능 이슈가 발생하지 않도록 집계 전략 설계가 필요 (e.g. 정각마다 통계 갱신, 쓰기 시 실시간 반영 등)
