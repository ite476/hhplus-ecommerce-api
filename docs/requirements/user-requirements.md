# 👤 User 도메인 요구사항

사용자 계정 관련 요구사항을 정의합니다.

---

## ✅ 기능 목록

### 1. 회원 가입 API

- 사용자 아이디(고유 문자열)를 입력 받아 회원 가입을 진행

---

## ⚠️ 정책 및 제약사항

### 1. 회원 가입 API
- 엔드포인트: `POST /join`
- 입력:
  - `header`
    - 사용자 식별자 `필수`
  - `body`
    - 충전 금액 `필수`
- 출력:
  - `성공` `201 Created`
  - `실패`
    - 이미 존재하는 아이디 `422 Unprocessable Entity`
    - 문자열 길이 제한 위반 `422 Unprocessable Entity`
      - 최소 : `2`
      - 최대 : `32`
