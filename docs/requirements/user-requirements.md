# 👤 User 도메인 요구사항

사용자 계정 관련 요구사항을 정의합니다.

---

## ✅ 기능 목록

### 1. 회원 가입 API

- 사용자 아이디(고유 문자열)를 입력 받아 회원 가입을 진행

---

## ⚠️ 정책 및 제약사항

- 회원 아이디는 ***임의 문자열을 입력***받아 가입 진행
  - 길이 제한 
    - 최소 `2` 
    - 최대 `32`
  - 입력 문자 유형 제한 
    - `영문 대소문자`
    - `숫자`

## 📘 API Spec

### 1. 회원 가입 API
> - 엔드포인트: `POST /join`
> - 입력:
>   - `header`
>     - 사용자 식별자 `필수`
>   - `body`
>     - 충전 금액 `필수`
> - 출력:
>   - `성공` `201 Created`
>   - `실패`
>     - 이미 존재하는 아이디 `422 Unprocessable Entity`
>     - 문자열 길이 제한 위반 `422 Unprocessable Entity`
