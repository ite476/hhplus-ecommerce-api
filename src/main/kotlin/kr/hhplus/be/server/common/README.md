# 🗂️ common

## 폴더 구조

```text
common/
├── config/              # Spring Configuration 클래스
│   ├── jpa/             # JPA/Hibernate 설정
│   ├── web/             # Web MVC, CORS 설정
│   └── security/        # Spring Security 설정
├── exception/           # 공통 예외 처리
├── response/            # 공통 응답 포맷
└── constants/           # 공통 상수
```

## 역할

**공통 설정 관리**: Spring Boot Configuration 및 크로스 커팅 관심사 처리
