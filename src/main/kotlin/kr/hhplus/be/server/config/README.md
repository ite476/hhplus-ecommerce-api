# config/

## 📌 역할
Spring Boot 애플리케이션 전반의 설정 클래스를 정의하는 디렉토리입니다.

- JPA 설정
- Swagger 설정
- Security 설정
- 기타 공통 설정 (ExceptionHandler, WebMvc 등)

---

## 🗂️ 디렉토리 구조 예시

```text
config/
├── jpa/
│ └── JpaConfig.kt
├── swagger/
│ └── SwaggerConfig.kt
├── security/
│ └── SecurityConfig.kt
└── ...
```
